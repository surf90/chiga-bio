# chiga-bio

茅ヶ崎の海の生き物がわかる



\# ちがビオ (chiga-bio) データ収集・編集ガイドライン



本ドキュメントは、「ちがビオ」のデータベース（`bio-data.json`）に収録する海洋生物データの収集方針、信頼できる情報源の選定基準、および著作権への配慮について定めたものです。利用者（ライフセーバーや一般の海浜利用者）の安全に直結するため、情報の「正確性」と「適法性」を最優先とします。



\## 1. データ収集の基本方針とJSON構造



各生物のデータは、医学的・科学的根拠に基づいた客観的な事実のみを抽出して構成します。特に「応急処置（FIRST\_AID）」については、民間療法を排除し、最新の医療ガイドラインに準拠してください。



収集・記述する主なJSONプロパティは以下の通りです：



\* \*\*`features` (特徴):\*\* \* 外見の特徴、生息環境、遭遇しやすい状況など、一般利用者が視覚的に判別できる情報を簡潔に記載する。

\* \*\*`firstAid` (応急処置):\*\* \* 受傷直後に行うべき物理的処置（洗浄、加温・冷却、圧迫など）をステップ順に記載する。最終的には「医療機関の受診」を推奨することを基本とする。

\* \*\*`dontDo` (禁忌事項):\*\* \* 「やってはいけないこと（例：真水で洗う、素手で触る、縛るなど）」とその医学的な理由を明記する。

\* \*\*`references` (出典):\*\* \* 記述の根拠となった文献の `title`, `author`, `year`, `doi` (またはURL) を必ず記載する。



\## 2. 推奨される情報源（ソース）



情報の裏付けとして、以下の公的機関、学術データベース、または専門機関が発行する査読付き論文・ガイドラインを一次情報として推奨します。



\### 国内の情報源

\* \*\*医療・救急プロトコル\*\*

&#x20;   \* 日本救急医学会 (JAAM): 毒のメカニズムや標準的な治療指針の確認。

&#x20;   \* 厚生労働省: 自然毒のリスクプロファイル等のデータ。

\* \*\*海洋・水産情報\*\*

&#x20;   \* 海上保安庁 海洋情報部: 一般向けの危険生物ガイドおよび発生状況。

&#x20;   \* 各都道府県の水産技術センター（例：神奈川県水産技術センター）: 相模湾等の局所的な生態動向や注意喚起。

\* \*\*学術論文データベース\*\*

&#x20;   \* J-STAGE: 国内の海洋生物学および臨床救急医学の査読付き論文。



\### 海外・国際的な情報源（グローバルスタンダード）

海外の公的機関は、海洋危険生物への対処ガイドラインが非常に充実しており、最新のエビデンスの確認に有用です。

\* \*\*NOAA (米国海洋大気庁):\*\* 海洋生物の生態および人との接触リスクに関する科学的データ。

\* \*\*DAN (Divers Alert Network):\*\* ダイバーやライフセーバー向けの、海洋生物による刺傷・咬傷に対する国際的なファーストエイド・ガイドライン。

\* \*\*CDC (米国疾病予防管理センター):\*\* 毒素に対する公衆衛生上の対応策。

\* \*\*WHO (世界保健機関):\*\* 国際的な毒物情報および標準的な公衆衛生ガイドライン。

\* \*\*ILS (国際ライフセービング連盟):\*\* 水辺の事故防止や応急処置に関する国際基準（Medical Position Statements）。



\## 3. 著作権・ライセンスへの配慮



「ちがビオ」はオープンかつ適法なプロジェクトとして運営するため、他者の著作権を厳格に尊重します。



\* \*\*事実と表現の分離（リライトの徹底）:\*\*

&#x20;   \* 生物の生態データや治療手順などの「事実」や「アイデア」そのものに著作権は発生しませんが、文献の「文章（表現）」には著作権があります。

&#x20;   \* \*\*禁止事項:\*\* 論文やウェブサイトの文章をそのままコピー＆ペーストすること。

&#x20;   \* \*\*必須事項:\*\* 複数の信頼できる情報源から事実関係を抽出し、\*\*必ず自分の言葉で、本アプリのフォーマット（箇条書き等）に合わせて要約・再構成（リライト）\*\*すること。

\* \*\*出典の明示:\*\*

&#x20;   \* リライトした場合であっても、情報の透明性と信頼性を担保するため、JSONの `references` 配列に出典元を必ず明記すること。可能な限りリンク（DOIや公式URL）を含める。

\* \*\*画像・アイコンの利用について (将来的な拡張時):\*\*

&#x20;   \* 他人の写真やイラストを無断使用しないこと。

&#x20;   \* 使用可能なライセンス：パブリックドメイン（CC0）、または クリエイティブ・コモンズ・ライセンス（CC BY 等、利用条件を満たすもの）。

&#x20;   \* 官公庁のオープンデータを利用する場合は、各機関の利用規約（政府標準利用規約など）に従い、適切にクレジット表記を行うこと。
