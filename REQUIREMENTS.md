# プロジェクト要件定義書: ちがビオ（chiga-bio）

## 1. プロジェクトの概要
湘南・茅ヶ崎の海辺において、ライフセーバーや一般の利用者が「遭遇した生き物」に関する科学的根拠に基づいた情報を即座に確認できる、非公式・海洋生物ガイドWebアプリを構築する。

## 2. 技術要件・制約
- **構成:** シングルページの静的HTML（Vanilla JS / CSS）。外部ライブラリへの依存は最小限とする。
- **デプロイ・ホスティング:** GitHub Pages。
- **CI/CD:** GitHub Actionsを利用した自動デプロイを想定。
- **拡張性:** 将来的なPWA（Progressive Web App）化を視野に入れ、電波状況の悪い海辺でのオフライン利用を可能にする構造とする。

## 3. ユーザー体験（UX）の定義と情報設計
- **視認性（屋外での利用）:** 強い直射日光下でも読みやすいコントラスト比。危険度は「アイコン＋文字」で表現し、色覚多様性に配慮する。
- **操作性（片手・濡れた手）:** 画面下部に主要なナビゲーションを固定し、親指の可動範囲で完結させる。タップ領域は最低48px以上を確保。
- **即時性:** 読み込み速度を最優先。緊急時に「刺されたらどうするか」がファーストビューですぐに分かるUI設計。

## 4. コンテンツ・編集方針
- **対象生物:** 茅ヶ崎海岸周辺で遭遇する「危険生物（カツオノエボシ、アカエイ等）」および「一般的な生物」。
- **信頼性の担保:**
  - 論文や公的機関（海上保安庁、日本救急医学会など）の情報をベースにリライトする。
  - 出典（DOIや論文名等）をアコーディオンメニュー内に確実に明記し、客観性を視覚的に担保する。
* **情報の階層化（修正）:**
    * **「ファースト・エイド（応急処置）」**、**「生物学的特徴（分類・生態）」**、**「茅ヶ崎での遭遇率・時期」**の3層構造でリライトし、緊急時と学習時のニーズを分ける。
* **「やってはいけないこと」の強調（追記）:**
    * 海洋生物被害では「酢をかける（種によって有効・有害が異なる）」「傷口を口で吸う」などの誤った俗説が多いため、日本救急医学会の知見に基づいた**「禁忌事項」**を赤字等で目立たせる。
- **著作権遵守:**
  - 画像はiNaturalist等の「パブリックドメイン（CC0）」または「CC BY（要クレジット表記）」の写真を厳選してリンクする。

## 5. ソース

### 1. 国内の公的機関・研究開発法人
* **環境省 (国家機関・省庁):** 海洋生物レッドリストや、国立公園内の生態系保護データを提供します。生物多様性保全の行政政策を担います。
* **厚生労働省 (国家機関・省庁):** 自然毒のリスクプロファイル等のデータを提供します。
* **水産庁 (国家機関・省庁):** 魚種ごとの資源量推移など、水産資源評価を提供します。漁業資源の持続可能な管理を担います。
* **海上保安庁 海洋情報部 (国家機関・省庁):** 一般向けの危険生物ガイドおよび発生状況を提供します。日本海洋データセンター (JODC) を運営し、海洋情報のハブを担います。海流、水温、海底地形、海洋汚染調査を行います。
* **JAMSTEC / 海洋研究開発機構 (国立研究開発法人):** 深海生物映像や深海生態系の専門研究データを提供します。世界トップクラスの深海調査を行っており、「BISMaL」などのデータベースもここに含まれます。
* **国立科学博物館 (独立行政法人):** 標本に基づく分類学的データや、魚類・海棲哺乳類の変遷に関するデータを提供します。自然史研究の基準（ナショナルコレクション）となります。
* **各都道府県の水産技術センター（例：神奈川県水産技術センター） (地方自治体):** 相模湾等の局所的な生態動向や注意喚起を確認できます。相模湾の赤潮・貝毒情報、近海魚類の出現動向も提供しています。

### 2. 国際的・欧米の公的機関および国際団体
* **WHO / 世界保健機関 (国連機関):** 国際的な毒物情報および標準的な公衆衛生ガイドラインを提供します。
* **UNESCO-IOC (国連機関):** 政府間海洋学委員会として海洋科学の国際連携を行う国連機関による世界的権威です。
* **CDC / 米国疾病予防管理センター (米国国家機関):** 毒素に対する公衆衛生上の対応策を提供します。
* **NOAA / 米国海洋大気庁 (米国国家機関):** 海洋生物の生態および人との接触リスクに関する科学的データを提供します。海洋環境モニタリングや海洋哺乳類保護など、科学的根拠に基づく高度な海洋政策を行います。
* **IUCN / 国際自然保護連合 (国家・政府機関が参加する国際ネットワーク):** 世界の絶滅危惧種の格付け（レッドリスト）を提供します。世界最大の自然保護ネットワークです。
* **ILS / 国際ライフセービング連盟 (国際NGO):** 水辺の事故防止や応急処置に関する国際基準（Medical Position Statements）を提供します。
* **DAN / Divers Alert Network (非営利団体):** ダイバーやライフセーバー向けの、海洋生物による刺傷・咬傷に対する国際的なファーストエイド・ガイドラインを提供します。

### 3. 国内の医療・救急・安全管理に関する団体
* **JLA / 日本ライフセービング協会 (公益財団法人):** 海辺の安全管理、全国的な事故統計と対策ガイドラインを提供します。
* **日本救急医学会 (学会・一般社団法人):** 毒のメカニズムや標準的な治療指針の確認に利用します。オニダルマオコゼやハブクラゲなど、毒を持つ海洋生物による刺咬傷の応急処置や臨床的知見を提供します。日本における救急医療の標準ガイドラインを策定しています。
* **日本臨床救急医学会 / 海洋曝露事故検討委員会 (学会・一般社団法人):** 救急医学会よりもさらに「現場での初動」に特化した報告書やプロトコルを提供します。

### 4. 国内の主要学会・専門データベース・民間機関
* **J-STAGE (国立研究開発法人JSTが運営するプラットフォーム):** 国内の海洋生物学および臨床救急医学の査読付き論文が検索できる学術論文データベースです。
* **日本海洋学会 (学会):** 海洋物理、化学、生物学を統合した広範な科学研究を行います。
* **日本魚類学会 / 日本貝類学会 / 日本ベントス学会 (学会):** 標準和名の確定、分類学的な最新知見、新種の記載を行います。
* **新江ノ島水族館 / えのすい (民間企業):** 相模湾の生物多様性に関する知見を提供します。特にクラゲ類の研究では世界屈指であり、カツオノエボシ等の生態情報は非常に正確です。

### 5. 世界標準の専門データベース
* **OBIS / Ocean Biodiversity Information System (UNESCO-IOC系の国際プロジェクト):** 海洋生物の出現地点を地図上にプロットした、分布調査の決定版です。
* **WoRMS / World Register of Marine Species (国際的な専門家ネットワーク):** 海洋生物の世界標準の学名を管理しており、学術資料作成には欠かせません。
* **FishBase / SeaLifeBase (国際的なコンソーシアム):** 生物の体長、食性、繁殖、分布域などを網羅した最大級の百科事典です。

---

 [!NOTE]
 **資料作成上の使い分け**
* **「どこに何がいるか」を調べる:** JAMSTEC (BISMaL) や OBIS。
* **「正しい名前や分類」を確認する:** 日本魚類学会 や WoRMS。
* **「危険生物への対策・治療プロトコル」を記述する:** 日本救急医学会、日本臨床救急医学会、DAN。
* **「生息域の海流や地形」を裏付ける:** 海上保安庁。
* **「茅ヶ崎・相模湾のリアルタイムな状況」を調べる:** 神奈川県水産技術センター。
* **「クラゲ類の正確な生態・同定」を調べる:** 新江ノ島水族館。
* **「ビーチでの安全管理・応急処置基準」を確認する:** JLA、ILS。
